🛒 JerseyHub – E-Commerce Jersey Store
📌 Project Overview

JerseyHub is a Django-based E-Commerce web application that allows users to browse, customize, and purchase football jerseys online.
The platform provides features such as user authentication, cart management, wishlist, customized player names and numbers, and order processing.

This project was developed as part of my Python Full Stack Development internship, where I implemented both frontend and backend functionalities using modern web technologies.
